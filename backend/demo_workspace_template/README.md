# RepoPilot Demo Workspace

Welcome! This is a **pre-loaded sample project** for exploring the IDE.

## What's inside

| Folder | Contents |
|--------|----------|
| `src/algorithms/` | Graph algorithms (Dijkstra, BFS, DFS) and sorting |
| `backend/` | Sample FastAPI service |
| `frontend/` | Sample TypeScript/React-style UI code |
| `docs/` | Architecture notes (try Markdown preview) |
| `tests/` | Pytest examples |
| `data/` | Large reference module for scrolling |

## Try these IDE features

1. **Explorer** — browse the file tree
2. **Ctrl+P** — quick open `graph.py` or `ARCHITECTURE.md`
3. **Ctrl+Shift+O** — jump to `dijkstra` or `merge_sort`
4. **Search** — find `TODO` or `Repilot` across files
5. **Outline** — view symbols in any Python/TS file
6. **Markdown preview** — open `docs/ARCHITECTURE.md` and click Preview
7. **Terminal** — run `python -m pytest tests/` or `dir src`
